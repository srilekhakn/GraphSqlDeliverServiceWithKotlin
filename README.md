Warehouse Delivery Management Service

Technologies/Tools:
-------------------
IDE: IntelliJ (STS/Eclipse)
Kotlin: 1.6.21
GraphQL & GraphiQL: 5.x.x
Build tool: Maven
Spring Boot: 2.6.7

-----------------------------

------------------------
java -jar assignment-0.0.1-SNAPSHOT.jar
